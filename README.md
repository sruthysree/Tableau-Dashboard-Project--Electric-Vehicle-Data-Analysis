# Tableau-Dashboard-Project--Electric-Vehicle-Data-Analysis
This project presents an end-to-end Tableau dashboard analyzing electric vehicle (EV) adoption across the United States. It transforms raw EV population data into interactive visualizations that highlight growth trends, regional distribution, manufacturer performance, and policy eligibility.

Output

Developed Tableau dashboard with dynamic filters, KPIs, and geographic mapping.
Clear visualization of EV growth trends by model year and state-level distribution.
Comparative analysis of top manufacturers and models (Tesla, Nissan, Chevrolet, Volkswagen, etc.).
Insights showing BEVs dominate with 77.6% share, Tesla leads with Model Y & Model 3 contributing 37%, and 41.8% of vehicles qualify for CAFV incentives.
